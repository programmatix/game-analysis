## Marvel Champions fonts

This folder is intentionally empty in git.

To use official Marvel Champions fonts in the generated PDFs:

- Put the `.ttf`/`.otf` files in this folder, or
- Pass `--fonts-dir /path/to/fonts`, or
- Pass `--font-config path/to/font-config.json` to map font keys to exact files.

`marvel-tuckbox` and `marvel-font-sheet` will fall back to Helvetica if fonts are missing.
